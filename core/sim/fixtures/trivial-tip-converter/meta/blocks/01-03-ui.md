## Context

01-02 lands the pure domain logic behind a genuine test suite but wires it into no UI. This block
builds the single-page form that makes the tip calculator and unit converter usable: bill/tip/party
inputs feeding `src/lib/tip.ts`, and miles/celsius inputs feeding `src/lib/convert.ts`, rendered
live on the page.

---

## Tasks

### T1: Converter form component

Add a client component (`src/app/converter-form.tsx`) with controlled inputs for bill amount, tip
percent, party size, miles, and Celsius. Compute and display: tip amount, total, per-person split,
km equivalent, and Fahrenheit equivalent — all via `src/lib/tip.ts` / `src/lib/convert.ts`, never
reimplemented in the component.

### T2: Wire into the page

Replace the scaffold placeholder in `src/app/page.tsx` with the converter form, keeping the page a
single screen (no routing, no additional pages).
